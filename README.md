# municipal-money-charts

## Generate inital municpal money chart with:

```js
MuniMoneyChart([muniData]).element
```

## Add data with:

```js
loadMunis([MuniData])
```
## Unload data with:

```js
unloadMunis()
```

## Highlight Muni:

```js
showMuni(id)
```

## Load Medians:

```js
loadMedians([medians])
```
